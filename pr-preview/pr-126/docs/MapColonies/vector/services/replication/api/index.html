replica-server (1.0.1)

Download OpenAPI specification:Download

License: MIT

A RESTful API for querying and saving geographic replicas (snapshot and deltas)

public

the endpoints which are exposed to the client

Get replicas by required and optional filters

Authorizations:
ApiKeyAuth
query Parameters
exclusive_from
string <date-time> (replicaDate)

The lower limit of filtering by date. the filter is exclusive.

to
string <date-time> (replicaDate)

The upper limit of filtering by date. the filter is exclusive.

sort
string
Default: "desc"
Enum: "asc" "desc"

The order in which the results are sorted based on their timestamp

geometry_type
required
string (geometryType)
Enum: "point" "linestring" "polygon"

return results with the following geometry type - point, linetrings or polygon

replica_type
required
string (replicaType)
Enum: "snapshot" "delta"

return replicas of the following type - snapshot, delta

layer_id
required
number (layerId)

return results from the following layer

Responses

https://mapcolonies.github.io/replica
