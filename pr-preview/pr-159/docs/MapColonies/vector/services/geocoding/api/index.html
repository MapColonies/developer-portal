Geocoding (0.2.2)

Download OpenAPI specification:Download

License: MIT

MapColonies Vector Geocoding api provides custom geodata search engine uniting multiple sources of data, query tiles, routes, items; Convertion functions - tile to/from WGS84 lat lng, WGS84 to/from US Army MGRS, WGS84 to/from UTM.

Location Name Based Search

Search engine

Search for geo-data. It serves data from multiple diverse mapcolonies vector sources and partners.

query Parameters
token
string

JWT authentication token provided by our team

query
required
string (Query) [ 3 .. 100 ] characters

Text to search

source
Array of strings (Source)

Sources to include (if not specified, all sources will be queried)

region
Array of strings (Region)

Regions to include (if not specified, all Regions will be queried)

object or WGS84Circle (object) or UTMCircle (object) (geo_context)

Geo context of search.

Supported queries:

  • bounding box:
    {"bbox":[minX,minY,maxX,maxY]}
  • By point and radius (WGS84 and UTM are supported)
    {"lon":value,"lat":value,"radius":value}
    {"x":value,"y":value,"zone": number, "radius":value}
geo_context_mode
string (geo_context_mode)
Enum: "filter" "bias"

Choose whether geo_context query parameter will be a filter or a bias value

limit
number [ 1 .. 15 ]
Default: 5

Maximum number of results

disable_fuzziness
boolean (disable_fuzziness)
Default: false

If an accurate result is obtained, only it will be returned

header Parameters
x-api-key
string

JWT authentication token provided by our team

Responses

https://mapcolonies.github.io/search/location/query

Control

Search tiles and sub tiles

Tiles are consisted of 3 characters.
You can query and get results from 2 characters.

If you define a tile (full name of it) you can query the sub tile associated with it.
You may also search tile based on 1 meter precision MGRS tile.

token
string

JWT authentication token provided by our team

tile
string [ 2 .. 3 ] characters

Tile name

sub_tile
string^[1-9][0-9]*$

Sub tile number

mgrs
string

1 meters MGRS Tile
